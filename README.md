# 📦 Ofis Takip — İndirmeler

> Personel · izin · departman · araç yönetim uygulaması.  
> Bu repo yalnızca derlenmiş `.exe` dosyalarını barındırır.

## ⬇️ İndir

**[→ Son Sürümü İndir](https://github.com/miraykk/ofis-takip-releases/releases/latest/download/OfisTakip.exe)**

veya tüm sürümler için: [Releases sayfası →](https://github.com/miraykk/ofis-takip-releases/releases)

---

## 🚀 Kurulum

1. Yukarıdaki bağlantıdan `OfisTakip.exe` dosyasını indir
2. İstediğin bir klasöre koy (kurulum gerektirmez)
3. Çift tıkla, çalıştır

> **Not:** İlk açılışta Windows SmartScreen uyarısı çıkabilir.  
> "Daha fazla bilgi" → "Yine de çalıştır" adımlarını izle.

---

## ✨ Özellikler

| Modül | Açıklama |
|---|---|
| 📅 Takvim | Aylık görünüm — izinler, doğum günleri, araç vadeler |
| 🕒 İzinler | Günlük / saatlik / yarım gün / geçmiş izin yönetimi |
| 👤 Personeller | Kişisel bilgi, TC maskeleme, bölge sorumlusu etiketi |
| 🏢 Departmanlar | Kart görünümü — personel & araç özeti |
| 🚗 Araçlar | Sigorta & muayene takibi, takvim entegrasyonu |
| 💼 Bordro Takip | Maaş & prim takibi |
| 🗂 Özlük Dosyası | Personel özlük belgeleri |
| 📝 Notlarım | Hatırlatıcılı not yönetimi |

## 💾 Verilerim Nerede?

Tüm veriler bilgisayarında yerel olarak saklanır, buluta gitmez:

```
%APPDATA%\OfisTakip\app.db
```

Bu dosyayı kopyalayarak verilerini yedekleyebilirsin.
